# Recipe Page

This project was built as part of the freeCodeCamp Responsive Web Design curriculum.

## Overview

This is a simple recipe webpage for Nigerian Jollof Rice created using HTML. The project demonstrates the use of semantic HTML elements to structure content clearly and accessibly.

## Technologies Used

- HTML5

## Features

- Recipe title and description
- Ingredients list
- Step-by-step cooking instructions
- Image with descriptive alternative text

## What I Learned

- HTML document structure
- Semantic HTML
- Ordered and unordered lists
- Images and accessibility
- Organizing webpage content
